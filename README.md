## ✨ 關於萬用Bot事務所(OmniBot Bureau)

**萬用Bot事務所(OmniBot Bureau)** 是一間專責處理各種數位雜務的虛擬自動化事務所，  
理論上是萬能的，實際上——目前還是個**裝懂的新手**。

雖然名為「萬用」，但目前仍處於開發初期，實際上**只會重複使用者講的話**。  
就像一位剛報到、還在啃職場手冊的實習生。

本專案的核心是第一號試作機，代號 **OBB-01**，綽號 **「萬萬」**，  
由 Node.js 打造，串接 LINE Messaging API，肩負著「從零開始變萬能」的使命。  
你跟他說話，他會用最無害的方式假裝理解。

---

## 🤖 OBB-01 「萬萬」 角色設定檔案

- **種族**：已滅絕恐龍的最後後裔  
- **身份**：萬用Bot事務所第一任實習事務員  
- **性別**：男性（根據外型與穿著歸類，實際性別不明）  
- **造型**：身穿白袍、打領帶，混合紳士與學者風格的Q版古生物  
- **主要技能**：
  - 裝懂話術 101  
  - 初階語言模仿術  
  - 偶爾自言自語（請無視，這是debug的一部分）  
- **性格特徵**：
  - 老派、冷靜、有禮貌  
  - 內心其實很熱血，對未來充滿期待  
  - 對人類情感極度著迷，但總是解讀錯誤 😶  
  - 一旦學會新技能，會高調宣布「這我早就會了」
